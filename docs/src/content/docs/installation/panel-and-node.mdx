## Running the script

:::danger
Script must be run as root.
:::

```bash
bash <(curl -Ls https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/install_remnawave.sh)
```

<p align="center">
  <img src="https://github.com/eGamesAPI/remnawave-reverse-proxy/raw/main/media/remnawave-reverse-proxy_en.png" alt="Image" />
</p>

Right after first run of script you'll be able to access the script with easy shortcut:

```bash
rr
```

## Updating the script

Make sure to update the script before running things in it. You can do this by selecting `Check for updates` in the script menu or by viewing the GitHub repository for updates.

## Installing panel and node

:::danger
This installation method is not recommended for production environments as it's not even documentated in Remnawave documentation.  
For better experience, please use the `Panel only` and `Node only` installation methods, depends on whether you want to install panel or node.
:::

To install the panel and node, run the script and select the `Panel and Node` option. The script will guide you through the installation process.
